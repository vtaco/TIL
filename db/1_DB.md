# DB

**체계화된 데이터의 모임**

몇개의 자료 파일을 조직적으로 통합하여 중복을 없애고 구조화하여 기억 시켜 놓은 자료의 집합체

- 중복 최소화
- 무결성
- 일관성
- 독립성
- 표준화
- 보안 유지

<br>

<br>

## RDB

**관계형 데이터 베이스**

- 스키마 : DB에서 자료구조, 표현 방법, 관계 등 전반적인 **명세를 기술**
- 테이블 : 열과 행의 모임을 사용해 조직된 데이터의 요소
- 열 : 고유한 형식이 저장됨
- 행 : 실제 데이터가 저장되는 형태
- PK : 각 행의 고유값
